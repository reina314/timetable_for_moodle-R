# Timetable for moodle+R

v0.1.0

[English version](https://github.com/reina314/timetable-for-moodle-R/blob/main/README.md)

Timetable for moodle+Rは非公式のブラウザ拡張機能で、受講科目を1週間の時間割表形式で表示することで**moodle+R** の利便性を改善します。

この拡張機能は、立命館大学のMoodleインスタンス（moodle+R）を利用する学生向けに設計されています。
授業スケジュール情報を抽出し、moodle+Rのダッシュボード上に直接、見やすい時間割表として表示します。

本プロジェクトは**非公式**であり、**Moodleや立命館大学とは一切関係ありません**。

------------------------------------------------------------------------

# 特徴

- **1週間の時間割表**をmoodle+Rのダッシュボードに直接表示
- 表示される項目：
  - 授業名
  - 教室/場所
  - 講師名
- 各時間割ブロックをクリックすることで、その講義の**コースページ**を閲覧可能
- 軽量設計でブラウザ上で直接動作

------------------------------------------------------------------------

# インストール方法

1.  このレポジトリをダウンロードまたはクローンします:

```bash
git clone https://github.com/reina314/timetable-for-moodle-R.git
```

## Chrome向け

2. Chromeを開き、次のリンクにアクセスします:

```
chrome://extensions
```

3. **Developer mode**を有効化します
4. **Load unpacked**をクリックします
5. `manifest.json`ファイルを選択します


## Firefox向け

2.  Firefoxを開き、次のリンクにアクセスします:

```
about:debugging
```

3.  **This Firefox**をクリックします
4.  **Load Temporary Add-on**をクリックします
5.  `manifest.json`ファイルを選択します

------------------------------------------------------------------------

# 使い方

1.  **moodle+R** にログインします
2.  **ダッシュボード** を開きます
3.  拡張機能が自動的に **1週間の時間割表** を追加します
4.  コースブロックにカーソルを合わせてハイライト表示します
5.  コースをクリックすると、moodle+R でコースページが開きます

moodle+R がコースイベントを読み込むたびに、時間割は自動的に更新されます。

# 必要な権限

この拡張機能は以下の権限を使用します：

### クッキー

認証済みAPIリクエストに必要なMoodleセッションクッキーを読み取るために使用されます。

### ストレージ

拡張機能の設定をローカルに保存します。

### ホスト権限

この拡張機能は次のホストとのみ通信します：

-   `*://lms.ritsumei.ac.jp/my/*`
-   `*://lms.ritsumei.ac.jp/lib/ajax/*`

データが外部のサーバに送信されることはありません。

------------------------------------------------------------------------

# プライバシー

-   個人データは収集されません。
-   追跡や分析は使用されません。
-   すべての処理はローカルのブラウザ内で行われます。

------------------------------------------------------------------------

# セキュリティ

この拡張機能は、moodle+Rデータのレンダリング時に`textContent`を使用することで、安全でないHTMLインジェクションを回避します。
これにより、悪意のあるコンテンツによるクロスサイトスクリプティングの脆弱性を防止します。

# 開発

プロジェクトの構成:

    .
    ├── manifest.json
    ├── content.js
    ├── background.js
    ├── style.css
    ├── README_ja.md
    └── README.md

主な構成要素:

-   **content.js** --- 時間割表UIをmoodle+Rページに挿入
-   **background.js** --- 安全なMoodle APIリクエストを実行
-   **style.css** --- 時間割表のレイアウトと視覚的スタイル
